LordNyriox's ReShade
====================
[![License](https://img.shields.io/badge/License-BSD_3--Clause-blue.svg)](https://opensource.org/licenses/BSD-3-Clause) [![MSBuild](https://github.com/LordNyriox/reshade/actions/workflows/msbuild.yml/badge.svg?branch=main&event=push)](https://github.com/LordNyriox/reshade/actions/workflows/msbuild.yml)

This is [**@LordNyriox**](https://github.com/LordNyriox)'s fork of ReShade, a generic post-processing injector for games and video software. Main changes are removal of Network Detection, disabling the splash screen at game start, and adding Github Actions.

## Building

Building of this fork is automated by a Github Actions workflow. Go [HERE](https://github.com/LordNyriox/reshade/actions/workflows/msbuild.yml) to download the latest build.
